# Student Performance Predictor

## Overview
This project predicts student marks based on study hours using Machine Learning (Linear Regression).

## Technologies Used
- Python
- Pandas
- Scikit-learn

## Features
- Reads dataset from CSV
- Splits data into training and testing sets
- Trains a Linear Regression model
- Predicts marks based on study hours
- Evaluates model performance using R² score

## Example

Input:
Enter study hours: 8.5

Output:
Predicted Marks: 85.0
